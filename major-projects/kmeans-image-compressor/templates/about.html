The Technology

Exploring the intersection of unsupervised machine learning, color theory, and data compression.

Structural vs. Visual Compression

Traditional image compression (like standard JPEG) works by analyzing frequencies and discarding visual information that the human eye is less sensitive to. It's a trick played on your vision.

This tool takes a fundamentally different approach. It doesn't blur details; it restructures the image's data. By analyzing the entire color space of an image, we use Unsupervised Machine Learning to rebuild the image using a drastically restricted, mathematically optimized palette of colors.

The result is an image with a significantly smaller file size that retains sharp edges and a distinct, often artistic, aesthetic.

The Engine: K-Means Clustering

The core of this tool is the K-Means algorithm. It is an algorithm used to find groups (clusters) in unlabelled data.

Imagine every pixel in your image is a point plotted in a 3D space (Red axis, Green axis, Blue axis). A colorful photo forms a dense cloud of millions of points.

How K-Means works here:

  1. You tell the algorithm how many colors you want in the final image (the value of K).
  2. The algorithm places K points randomly into that 3D color cloud.
  3. It iteratively moves those points until they sit in the exact gravitational center of the densest clusters of colors.

The final image is reconstructed by snapping every original pixel to its nearest optimized center color.

Intelligent Automation (Entropy)

The hardest part of palette reduction is choosing the right number of colors ($K$). Choose too few, and the image looks posterized and banded. Choose too many, and the file size remains large.

Our "Auto K" feature utilizes concepts from Information Theory (specifically Shannon Entropy).

A flat blue sky has low entropy; a dense forest photo has high entropy. We use this score to intelligently map the image to an optimal $K$ value that balances visual fidelity against file size savings, without human guesswork.
