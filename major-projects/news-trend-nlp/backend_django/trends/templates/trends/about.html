About Project

This project demonstrates a full-stack NLP pipeline deployed on Render's Free Tier.

Tech Stack

  • Django: Backend & API
  • FastAPI: Topic Modeling Service (Scikit-Learn NMF)
  • GDELT Project: Source of global news data
  • LangChain: Summarization via HuggingFace Inference API

Architecture

We use a microservice architecture to separate the heavy ML processing (FastAPI) from the web application (Django). This ensures better resource management within limited container memory.
